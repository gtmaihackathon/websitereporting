# Productsup — Monthly SEO Performance Report

Interactive SEO performance dashboard (Organic & Direct) combining
Google Analytics 4, Google Search Console, and Semrush data.

**🔗 Live dashboard:** https://<your-username>.github.io/productsup-seo-report/

## What's inside
- **Overview** — headline KPIs and top priorities
- **Rank Tracking (Semrush)** — tag scorecard, quick wins, striking distance, AI Overview wins
- **Search Console** — clicks, impressions, position, top queries & pages
- **Competitors** — benchmark and keyword gaps
- **Traffic & Quality (GA4)** — sessions, engagement, drop/gain by page & country
- **Pages & Conversions** — which pages drive which key events
- **Pipeline · SQO** — funnel, channel/region performance, content that converts
- **Actions & Calendar** — recommendations and reporting schedule

## Reporting period
Currently showing **April–May 2026**. Data is a point-in-time snapshot,
refreshed monthly.

## How to update (monthly)
1. Generate the new dashboard file.
2. Rename it to `index.html`.
3. Upload it to this repo (replacing the old one) and commit.
4. The live link above updates automatically.
